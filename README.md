# To-Do List App

This is a clean, simple task management app I built to master vanilla JavaScript, DOM manipulation, and data persistence. It's designed to be lightweight, easy to use, and completely distraction-free.

## 🔗 Links

- **Live Demo:** [Check out the live site](https://todolistappfswd.netlify.app/)
- **Code Repository:** [View the GitHub Repository](https://github.com/MadalitsoB)

---

## ✨ Features

- **Manage Tasks:** Easily add new tasks, check them off when completed, or delete them when they're no longer needed.
- **Filter Views:** Quickly switch between *All*, *Active*, and *Completed* tasks to stay on top of your daily goals.
- **Data Persistence:** I integrated `localStorage` so that your tasks don't disappear when you refresh the page or close your browser tab.
- **Responsive Design:** A simple, modern layout that looks and works great on both desktop and mobile screens.

## 🎯 The Challenge & Constraints

Compared to the static layout projects I built last term, this application was a massive step up. It pushed me to focus heavily on JavaScript and interactive logic.

To make things even more interesting, this project was built under a strict learning constraint: **no video tutorials, no YouTube code-alongs, and no copy-pasting from Stack Overflow.** Instead, I had to rely 100% on reading the official **MDN Web Docs**, testing code in my browser, and debugging issues through trial and error. This really taught me how to solve problems like a real developer.

## 💡 What I Learned From This Project

Building this app from scratch helped me truly understand how JavaScript interacts with the browser. Some of my biggest takeaways include:

- **DOM Manipulation:** Getting comfortable with selecting elements dynamically using `.querySelector()`, creating new HTML items with `.createElement()`, and rendering them on the page.
- **Handling Events:** Learning how to listen for user actions (like mouse clicks or hitting the Enter key) and making the app respond smoothly.
- **Working with localStorage:** Figuring out how to convert data into strings using `JSON.stringify()` to save it, and then parsing it back into JavaScript when the page reloads.
- **Problem Solving:** Learning not to panic when my code broke, and instead breaking the problem down into smaller steps to find and fix the bugs.
